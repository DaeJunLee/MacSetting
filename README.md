# MacSetting

## HomeBrew Install
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

- In iterm
echo export PATH="/usr/local/bin:$PATH" >> ~/.zshrc

- In Commone line
echo export PATH="/usr/local/bin:$PATH" >> ~/.bash_profile

```

## Node Intall
```
brew intall node

- Confirm node and npm
node -v
npm -v
```
